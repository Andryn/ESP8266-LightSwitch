# ESP8266-LightSwitch
2-channel ESP8266 based lightswitch for ESP8266 ESP-12 Full Evaluation Board
Work on NodeMCU 0.9.6 https://github.com/nodemcu/nodemcu-firmware/releases/tag/0.9.6-dev_20150406
